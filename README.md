# 🏞️ TadobaLand Backend

Backend API for **TadobaLand** — a niche land marketplace focused on property deals around Tadoba (lease, sale, resort development opportunities).

Built with a scalable architecture using **Node.js, Express, Prisma, and PostgreSQL**.

---

## 🚀 Features

- 🔐 Authentication (Signup / Login)
- 🏡 Property Listing (Owner / Agent)
- 🔍 Advanced Property Search & Filters
- 📸 Property Image Handling (planned)
- 📩 Lead Generation System (Buyer → Owner contact)
- 💳 Lead Credit System (monetization ready)
- 🧩 Role-based system (OWNER, AGENT, BUYER, ADMIN)

---

## 🏗️ Tech Stack

- **Backend:** Node.js, Express
- **Database:** PostgreSQL
- **ORM:** Prisma
- **Auth:** JWT (via vedaxon-auth-framework)
- **Deployment:** Render (planned)
